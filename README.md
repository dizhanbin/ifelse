# ifelse 
    android 辅助开发插件.
## 功能、目标
    界面、业务统一可视化管理
    复杂业务可视化，减少代码中的if else 判断
    UI与业务一体化集成，实现一个业务处理的连贯性
    生成代码可配置、模版化
